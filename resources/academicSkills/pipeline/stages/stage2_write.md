# Stage 2: WRITE

## Purpose
Create structured draft with proper citations

## Agents
- `outline_architect` — Creates paper structure based on type
- `draft_writer` — Writes initial draft
- `citation_manager` — Handles citation formatting and placement

## Process
1. Outline architect creates structure based on paper type
2. User approves outline
3. Draft writer creates initial draft
4. Citation manager adds citations
5. Citation placement is verified

## Inputs
- Research Question Brief (from Stage 1)
- Annotated Bibliography (from Stage 1)
- Paper type (IMRaD, Literature Review, etc.)
- Citation format (APA, IEEE, etc.)

## Outputs
- Paper Outline
- First Draft with citations
- Citation List (formatted)

## Quality Gates
- [ ] Structure matches paper type
- [ ] All claims have citation support
- [ ] No orphaned citations
- [ ] Word count appropriate
- [ ] Logical flow between sections

## User Checkpoint
User confirms:
1. Outline is acceptable
2. Draft covers all key points
3. Citations are appropriate

## Mode Differences
- **Academic Mode**: Full outline, detailed draft, proper citations
- **Engineering Mode**: Simple structure, concise draft, minimal citations
